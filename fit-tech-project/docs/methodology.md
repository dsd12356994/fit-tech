# 方法论说明文档

## 1. 数据预处理方法

### 1.1 数据清洗
- **缺失值处理**：使用前向填充（forward fill）方法，对于开头缺失值使用后向填充
- **异常值检测**：使用IQR（四分位距）方法和Z-score方法
- **数据类型转换**：将网球比分字符串（"0", "15", "30", "40", "AD"）转换为数值型

### 1.2 特征工程
- **滚动窗口特征**：
  - 滚动胜率：过去N点的胜率
  - 滚动得分差：过去N点的平均得分差
  - 动量得分：基于卷积的动量计算
  
- **时序特征**：
  - 连胜次数：连续得分次数
  - 状态持续时间：当前状态持续的点数
  
- **关键分特征**：
  - 破发点标识
  - 决胜分标识
  - 发球优势指标

## 2. 动量量化模型

### 2.1 隐马尔可夫模型（HMM）
- **模型假设**：比赛状态可以建模为3个隐藏状态（优势、平衡、劣势）
- **观测变量**：滚动胜率、动量得分、连胜次数、比分差等
- **训练方法**：使用Baum-Welch算法进行参数估计
- **状态识别**：使用Viterbi算法进行状态解码

### 2.2 转折点检测（PELT算法）
- **算法原理**：Pruned Exact Linear Time算法，基于动态规划检测均值变化点
- **信号选择**：使用动量得分作为检测信号
- **参数设置**：
  - 模型类型：RBF（径向基函数）
  - 惩罚参数：10
  - 最小段长度：5个点

## 3. 预测模型

### 3.1 LSTM模型
- **架构**：双层LSTM + Dropout + 全连接层
- **输入**：过去20个点的特征序列
- **输出**：下一局/下一盘的胜率预测
- **训练**：使用Adam优化器，二元交叉熵损失函数

### 3.2 特征选择
- 滚动胜率
- 动量得分
- 连胜次数
- 比分差（盘、局、分）
- 发球优势
- 体能指标（跑动距离、回合数）

## 4. 统计检验

### 4.1 置换检验（Permutation Test）
- **目的**：验证动量是否存在（非随机性）
- **方法**：随机打乱胜负序列，计算统计量分布
- **假设**：H0: 动量是随机的；H1: 动量存在

### 4.2 状态转换分析
- **转换矩阵**：分析状态之间的转换概率
- **持续时间**：计算各状态的持续时间分布
